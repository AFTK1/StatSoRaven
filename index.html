StatSoRaven

A data visualization site for the Baltimore Ravens

This project is about creating data visualizations for the Baltimore Ravens team as well as their individual players. The main motivating reason for choosing this is simply because it is our favorite NFL team. In the world of sports, it is common to discuss stats amongst each other so we wanted to create visualizations that would help facilitate this kind of discussion as well as show how the Baltimore Ravens have improved, or not, over time.


The main questions we want to answer with this project are:

• Have the Ravens improved/digressed throughout time?

• Do the Ravens have specific strengths or weaknesses?

• How do individual players’ stats compare to themselves?

• Have the players themselves improved/digressed throughout time?
